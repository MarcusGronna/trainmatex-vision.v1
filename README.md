# TrainMateX-Vision

> 🎓 A learning project built during a **< SALT /> hackweek**.

## About

TrainMateX-Vision is a full-stack web application created as a hackweek project at [< SALT />](https://salt.dev). The primary goal of this project is to learn and explore modern web technologies in a hands-on, collaborative setting.

## Tech Stack

- **Frontend** – React 19, TanStack Router & Query, Tailwind CSS, Clerk (auth), Vite
- **Backend** – .NET (ASP.NET Core Web API)

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

The frontend dev server runs on port **3000** by default.

### Backend

```bash
cd backend/TrainMateX.Api
dotnet run
```

## Project Structure

```
├── frontend/          # React SPA (Vite + TanStack)
├── backend/
│   └── TrainMateX.Api # .NET Web API
└── TrainMateX-Vision.sln
```
